# Calendar Generator

A monthly calendar generator built with PHP and a modern, responsive UI.

## Features

- Month and year selection.
- Year validation between 1990 and 2030.
- Monthly calendar with:
  - Weekdays.
  - Weekends.
  - Fixed public holidays.
  - Holidays that overlap with weekends.
- Aquamarine-to-emerald visual theme.

## Requirements

- PHP 7.4 or higher.
- XAMPP (recommended) or any PHP-enabled web server.

## Run Locally (XAMPP)

1. Copy the project folder into `htdocs`.
2. Start Apache from the XAMPP control panel.
3. Open in your browser:

   `http://localhost/php_calendario/calendario.php`

## Project Structure

- `calendario.php`: calendar generation logic and UI styling.

## Notes

- This project uses fixed holidays defined in the source code.
- No database or external dependencies are required.
